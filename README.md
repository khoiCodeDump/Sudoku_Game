# Sudoku_Game

The game is all coded by me Khoi Vu. 

The game is fully working with a GUI implemented using Java Swing. To play the game, please download the JAR file and run the file.
A simple UI will be presented and you will have the option to play at 3 difficulties. 
Easy will have 32 blank spots, medium will have 34 blank spots, hard will have 36 blank spots.
Following the official definition of Sudoku game, the game is generated with a guaranteed unique solution, meaning there is only one solution. 
Sudoku.java generates the board game on the fly with backtracking algorithm. 
No base maps were used for map generation, thus the time and compute power to generate a valid map will vary.
